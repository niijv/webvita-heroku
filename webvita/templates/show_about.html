About

... me

My name is Elias Zeitfogel and I am, among many other things, interested in computer graphics / vision, social network analysis, machine learning, data science, web development and learning new skills. In my free time I like to do obstacle races and go mountain biking.

I graduated from the University of Technology Graz where I studied Computer Science and Software Development and Business Management. Some of my work there has been published in the paper Evolution of Reddit: From the Front Page of the Internet to a Self-referential Community? at the World Wide Web Conference 2014 in Seoul, Korea.

I also do a lot of freelancing, some projects I worked on can be viewed here.

... this website

After finishing my studies I finally found time to dive into Flask, a Python microframework, and this website is the result of that endeavour. I use it to write about my experience while working on new projects and for some other stuff I deem interesting enough to share. Additionally some of my recent projects and accomplishments are presented in the Reference aswell, hence the clever name "Webvita". This website will continue to grow whenever I find the time to add new features or tweak existing ones.

The website was created with Flask (including various extensions) and some Bootstrap and is currently hosted on a free tier of OpenShift. You are free to fork it on GitHub or use the version I prepared for Heroku.
